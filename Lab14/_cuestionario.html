Cuestionario

1. ¿Qué es ODBC y para qué es útil?
Open Database Connecitivity es un estandar desarrollado por SQL Access Group, es útil para acceder a cualquier dato desde cualquier aplicación, sin importar qué DBMS almacene los datos.



2. ¿Qué es SQL Injection?
Es una técnica de hackeo donde como su nombre de usuario/ID de usuario, y en lugar de un nombre/ID, el usuario le da una declaración SQL que ejecutará sin saberlo en su base de datos.



3. ¿¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?
Para evitar los ataques se puede usar parámetros SQL. Los parámetros SQL son valores que se agregan a una consulta SQL en tiempo de ejecución, de forma controlada.
