Cuestionario

  • 1. ¿A qué se refiere la descentralización de servicios web?
    Se refiere a tener servicios web desplegados en diferentes lugares o sitios.
  • 2. ¿Cómo puede implementarse un entorno con servicios web disponibles aún cuando falle un servidor?
    Descentralizando los servicios web para que cuando falle un servidor se pueda obtener el servicio desde otro servidor.
